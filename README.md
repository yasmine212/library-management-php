# Library Management System (PHP)

This project allows basic library operations such as managing books and users.

## Technologies
- PHP
- MySQL
- HTML / CSS
- JS

## How to Run

1. Install XAMPP
2. Place the project in:
   C:\xampp\htdocs\library-system

3. Start Apache and MySQL

4. Import the database:
   - Open http://localhost/phpmyadmin
   - Create a database (e.g. library_db)
   - Import `database.sql`

5. Run the project:
   http://localhost/library-system

- Developed as a team academic project
